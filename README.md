# 📋 Task Management System

A web-based **Task Management System** designed to help users efficiently create, manage, and track tasks. This system improves productivity by organizing workflows, assigning responsibilities, and monitoring task progress.

---

## 🚀 Features

- ✅ User Authentication (Login/Register)
- 📝 Create, Update, Delete Tasks (CRUD)
- 📊 Task Status Tracking (Pending, In Progress, Completed)
- 👥 Role-Based Access (Admin/User)
- 📅 Task Deadlines & Scheduling
- 🔍 Search & Filter Tasks
- 📈 Dashboard Overview
- 📁 Organized Task Lists
